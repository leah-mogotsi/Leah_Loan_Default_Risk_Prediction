# Loan Default Risk Prediction System

A simple machine learning web application that predicts whether a loan applicant is likely to default on a loan. The system uses a trained model and an interactive Streamlit interface to simulate how financial institutions assess borrower risk.

## LIVE APPLICATION

Try the deployed app here:
**https://leahloandefaultriskprediction-jgmznqrschzjxyw3krg63a.streamlit.app/**

## PROJECT OVERVIEW

This project demonstrates how data science and machine learning can assist financial institutions in evaluating loan applicants. By entering customer details such as income, employment history, and ownership status, the system predicts a risk flag:

| Flag | Meaning |
|------|---------|
| `0`  | Low default risk — proceed with approval |
| `1`  | High default risk — decline or escalate |
